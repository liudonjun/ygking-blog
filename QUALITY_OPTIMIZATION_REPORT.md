## 🔍 技术实现详情

### 自动化修复脚本
1. **批量 frontmatter 修复** (Python 脚本):
   ```python
   # tag → tags 字段重命名
   re.sub(r'^(tag:)\s*$', r'tags:', content, flags=re.MULTILINE)
   ```

2. **日期提取** (Git 历史分析):
   ```bash
   git log --diff-filter=A --format=%ai -- "*.md"
   ```

3. **分类自动推断** (基于目录结构):
   ```python
   category_mapping = {
       'dart': ['Dart'],
       'flutter': ['Flutter'], 
       'kt': ['Kotlin', 'Android'],
       'git': ['Git']
   }
   ```

4. **标题层级修复**:
   ```python
   # 将 body 中的 h1 降级为 h2
   re.sub(r'^# ([^#])', r'## \1', body, flags=re.MULTILINE)
   ```

### 验证方法
- 使用正则表达式精确匹配 frontmatter 格式
- 代码块状态机分析确保正确识别开始/结束标签
- Git 历史验证日期准确性
- 目录结构验证分类推断
