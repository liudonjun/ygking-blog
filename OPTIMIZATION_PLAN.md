# YGKing Blog 优化计划

## 项目概述
这是一个基于 VitePress 的技术博客项目，包含以下内容：
- Kotlin 系列文章（AndroidKotlin + KotlinJS）
- Flutter 鸿蒙化系列
- Dart 基础系列
- Git 版本控制指南

## 优化目标

### 1. 代码结构优化
- 统一目录命名规范（使用连字符而非下划线）
- 建立清晰的层级结构
- 优化文件组织方式

### 2. 文档格式标准化
- 统一 Markdown 格式（标题层级、代码块样式）
- 规范元数据格式（frontmatter）
- 统一图片引用方式

### 3. 内容优化
- 补充缺失的系列文章
- 改进文章质量
- 添加交叉引用链接

### 4. 配置优化
- 优化 VitePress 配置
- 改进导航和侧边栏
- 优化搜索功能

### 5. 链接和引用检查
- 修复损坏的内部链接
- 验证外部链接有效性
- 优化交叉引用

## 执行策略

使用多个 Claude Code 子代理并行处理不同优化任务：

1. **结构优化代理** - 重组目录结构
2. **格式标准化代理** - 统一文档格式
3. **内容优化代理** - 改进文章内容
4. **配置优化代理** - 优化 VitePress 配置
5. **链接检查代理** - 验证和修复链接

每个代理将独立工作，最后汇总结果。
